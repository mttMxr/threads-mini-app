🔗 Подключение Threads API

Для работы с Threads API необходимо получить Access Token от Meta Developer Console.

💡 Как получить токен:
1. Перейдите на Facebook Developers
2. Создайте приложение → Добавьте продукт "Threads API"
3. В настройках приложения получите Access Token
4. User ID можно найти в профиле Facebook
5. ВАЖНО: Добавьте разрешения: "threads_basic", "threads_content_publish"
